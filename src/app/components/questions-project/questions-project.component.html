Frequently Asked Questions Project

Angular Single Page App that displays a list of questions and answers. Angular App allows a user to add, edit, or delete a question/answer pair from the list of questions and answers. A user enters a question and answer and then behind the scenes the Angular App uses an Angular Service and Local Storage to store question/answer pairs.
  • Angular, Angular Components, Angular Services, Angular Directives, Angular 2-Way Data Binding, Angular Inputs, Angular Outputs, Angular CLI, Local Storage, TypeScript, TypeScript Models, node.js, npm, Bootstrap 4, CSS3
