Actors Project

Angular Single Page App that uses an Angular Service, RxJS, an Observable, and a JSON File to populate an Angular Component HTML Template. Angular App consists of one Angular Service and two Angular Components. Each Angular Component uses the Angular Service to get data from a JSON file. Each Angular Component subscribes to the observable response returned from the Angular Service. Each Angular Component uses the *ngFor directive and the actors property to render an unordered list inside each Angular Component HTML Template.
  • Angular, Angular Components, Angular Services, Angular CLI, RxJS, Observables, TypeScript, node.js, npm, Bootstrap 4, CSS3
