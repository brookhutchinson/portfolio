Firebase Authentication Service Project

Angular Single Page App that uses the Firebase Authentication Service to allow a user to login to an Angular App using their Google User Account Credentials. To login to the Angular App a user must first click the Login Button and then enter their Google Account Email and Google Account Password. An Angular Service calls the Firebase Authentication Service to authenticate the Google User Account Credentials. If a user has successfully been authenticated, the Angular Router will navigate the user to the HomeComponent. If a user has not been authenticated, the Angular Router will navigate the user to the LoginComponent. Once a user has been authenticatetd, a user can click the Logout Button inside the HomeComponent HTML Template to logout of the Angular App. When a user is logged out, the Angular Router will navigate the user to the LoginComponent.
