Angular Router Project

Angular Single Page App that uses the Angular Router and a Navbar Component to allow the user to navigate between two Angular Component Views. Angular App consists of multiple Angular Components. The Navbar Component and the Jumbotron Component each contain properties that are rendered inside HTML Templates using string interpolation and data binding expressions. A user can click a menu item link inside the Navbar Component to navigate to either the Home Component HTML Template or About Component HTML Template.
  • Angular, Angular Router, Angular Components, Angular CLI, TypeScript, node.js, npm, Bootstrap 4, CSS3
