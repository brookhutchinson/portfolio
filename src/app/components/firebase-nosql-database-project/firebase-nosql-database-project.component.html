Firebase Real-Time NoSQL Database Project

Angular Single Page App that connects to a Firebase NoSQL Database and displays property values inside an Angular HTML Template. Angular App uses Google Firebase Services, a NoSQL Database, the AngularFire2 Module, RxJS, and an Observable Property. The Firebase NoSQL Database contains a collection of 9 items. Each item contains a property name and a property value. An Angular Component uses string interpolation and data binding expresssions to render each of the Firebase Database item values inside the Angular Component HTML Template. Anytime data is changed inside the Firebase NoSQL Database the Firebase NoSQL Database updates the HomeComponent HTML Template in real-time to show the updated data item values.
