Github Web API Project

Angular Single Page App that allows a user to search for Github Profile Info and Github Repo Info by a Github Username. A user enters a Github Username and behind the scenes the Angular App uses an Angular Service, RxJS, and Observables to communicate with the Github Web API and retrieve Github Profile Info and Github Repo Info for the specific Github Username. A user can enter any valid Github Username. Example Github Usernames are "Google", "Netflix", "Hulu" and "Twitter".
  • Angular, Angular Components, Angular Services, RxJS, Observables, Github Web API, Angular Router, Angular Directives, Angular 2-Way Data Binding, Angular CLI, TypeScript, TypeScript Models, node.js, npm, Bootstrap 4, CSS3
