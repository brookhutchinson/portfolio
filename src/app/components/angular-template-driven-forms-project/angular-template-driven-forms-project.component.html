Angular Template-Driven Forms Project

Angular Single Page App that allows a user to enter information about a book into a form. Each form input control has different user validation rules. Title is required and must be between 5 and 30 characters long. Author is required. URL is required and must be a valid URL beginning with either http:// or https:// Uses Angular Template-Driven Forms, Angular Form Template Variables, the ngModel Angular directive, and HTML5 validation attributes to check the user entered values and determine whether to show an alert message to the user. Uses Angular Form-Specific CSS Classes to tell the user if the form input control value is a valid value by changing the color of the left border of the form input control. A green border means a valid value. A red border means an invalid value. Uses Angular 2-Way Data Binding to show the TypeScript Model Object property values at the bottom of the HTML Template.
