Github Web API Project

Angular Single Page App that allows a user to enter a product name and filter a product list. Allows a user to navigate between views. Behind the scenes the Angular App uses multiple Angular Modules, multiple Angular Components, Angular Service, RxJS, and Observables to communicate with a JSON file that contains a list of products. Multiple routes are defined. The ProductDetailComponent route has a route guard that prevents a user from navigating from the ProductListComponent to the ProductDetailComponent if the product that is defined as a route parameter is not greater than zero or is not a number.
  • Angular, Angular Components, Angular Services, Angular Router, RxJS, Observables, JSON, Angular 2-Way Data Binding, Angular CLI, TypeScript, Interfaces, node.js, npm, Bootstrap 4, Font-Awesome, SCSS
