Sign In Landing Page

Sign In Landing Page for a Company Website. Uses the Bootstrap CSS Component Library to incorporate responsive design features. Includes a transition for showing a Sign In Modal Form, Form Placeholders for Username and Password, and links to pages if a user forgets their username or password. Displays one of three rotating background images. Uses CSS Rules and Media Queries to replace the background image based on the device size, position elements on the page, and define colors for elements on the page. When a user refreshes the browser, the page randomly selects one of three background images to use for the background image of the page.
  • Bootstrap 3, CSS3, CSS Media Queries, Ghost Buttons, JavaScript, jQuery, Placeholder.js
